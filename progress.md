Install perl: 
     wget http://coredev.nl/cydia/coredev.pub
     apt-key add coredev.pub
     echo 'deb http://coredev.nl/cydia iphone main' > /etc/apt/sources.list.d/coredev.nl.list
     apt-get update
     apt-get install perl
